About this website

Let There Be Bite is an AI-powered recipe generator that helps you find something interesting to cook for every meal.

This project has the following features:

  • AI prompts customised according to your recipe selection type
  • personalised recipes generated with variety in ingredients and cuisines
  • copy, share or download your recipes to save them, and
  • clean, aesthetic UI that works on desktop and mobile.

Front-end: Built with basic HTML, CSS, and JavaScript.
Back-end: Powered by Groq to handle prompt input and recipe generation, and Spoonacular for nutrition info. Backend logic is hosted on Render.
